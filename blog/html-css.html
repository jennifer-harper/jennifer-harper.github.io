What is the difference between Margin, Border, and Padding?

Margin, Border and Padding are used for the visual styling of elements. They added distance, space and/or style to an object.
In a nutshell margins are used to create space between elements, from the outside of an elements border. Padding is used to create space within an element and to separate content within that element. Borders are used as a visual identifier and unlike margin and padding are not designed to separate elements from each other.
